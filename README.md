# Public-chat

Real-Time Chat Application Using Socket.io.



## Run Locally

Clone the project

```bash
  git clone https://github.com/TANTANMOY/public-chat.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
